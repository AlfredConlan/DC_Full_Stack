## Featured stack

    -Node
    -Express
    -Sequelize
    -PostgreSQL
    -bcrypt
    -Express ES6 Template Engine

## Install

    git clone https://github.com/Jakejenk/DC_Full_Stack.git

    npm install (installs all dependencies)

## Run

    npm start
